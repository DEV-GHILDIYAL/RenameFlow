# 🔄 RenameFlow

**Bulk Image Renaming & WEBP Conversion — 100% Client-Side**

RenameFlow is a fast, privacy-first web application for bulk renaming images with SEO-friendly filenames and converting them to WEBP format. Everything runs in your browser — no server, no uploads, no data leaves your machine.

---

## ✨ Features

### 📂 Smart Upload
- **Drag & drop** multiple images at once
- **Click to browse** files from your filesystem
- Supports JPG, PNG, GIF, BMP, TIFF, SVG, and WEBP inputs
- Handles **100+ images** with optimized performance

### 📊 Dynamic Naming Table
- **Custom naming columns** — create columns like *Client Name*, *Page Name*, *Section Name*, *Image Type*, etc.
- **Editable inline cells** — type values directly in the table
- **Live filename preview** — see the generated SEO-friendly filename update in real-time
- **Bulk fill** — fill an entire column with the same value in one click
- **Quick preset columns** — one-click to add common column types
- **Add/remove/rename columns** on the fly

### 🏷️ SEO-Friendly Filename Generation
The final filename is auto-generated by combining all column values with these rules:

| # | Rule | Example |
|---|------|---------|
| 1 | Convert to lowercase | `A&B Builders` → `a&b builders` |
| 2 | Replace `&` with `and` | `a&b builders` → `a-and-b builders` |
| 3 | Replace spaces with hyphens | `a-and-b builders` → `a-and-b-builders` |
| 4 | Remove special characters | `hello@world!` → `helloworld` |
| 5 | Collapse duplicate hyphens | `a---b` → `a-b` |
| 6 | Trim leading/trailing hyphens | `-hello-` → `hello` |
| 7 | Skip empty fields | Empty columns are ignored |
| 8 | Append `.webp` extension | `final-name` → `final-name.webp` |

**Example:**
```
Client Name: A&B Builders
Page Name:   Home Page
Section:     Hero Banner

Result: a-and-b-builders-home-page-hero-banner.webp
```

### 🔄 WEBP Conversion
- Converts all image formats to **WEBP** using the **Canvas API**
- High-quality output (0.85 quality factor)
- **Batch processing** with visual progress bar
- Processes in batches of 5 to prevent browser memory issues

### 📦 ZIP Download
- All converted images bundled into a single **ZIP file**
- Powered by **JSZip** for reliable client-side archiving
- One-click download after processing

### 🛡️ Duplicate Handling
- Automatically detects duplicate filenames
- Appends `-2`, `-3`, etc. to ensure uniqueness
- Deduplication runs live as you type

## 🚀 Getting Started

### Prerequisites
- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/RenameFlow.git
cd RenameFlow

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist/` directory. Deploy to any static hosting (Netlify, Vercel, GitHub Pages, S3, etc.).

### Preview Production Build

```bash
npm run preview
```

## 🎯 Usage Guide

### Step 1: Upload Images
Drag and drop images onto the upload area, or click to browse files. You can add more images at any time.

### Step 2: Create Naming Columns
Click the **quick preset buttons** (Client Name, Page Name, etc.) or type a custom column name and click **Add Column**. You can:
- **Double-click** a column tag to rename it
- **Hover and click ×** to remove a column
- Add as many columns as needed

### Step 3: Fill in Values
Type values in each cell of the table. Use the **bulk fill button** (📋 icon in column headers) to fill an entire column with the same value.

### Step 4: Review Filenames
Watch the **Result Filename** column update in real-time. The generated filenames follow SEO best practices with lowercase, hyphenated, clean naming.

### Step 5: Apply & Download
1. Click **Apply Rename & Convert** to start WEBP conversion
2. Watch the progress bar
3. Click **Download ZIP** to get all your renamed WEBP images
